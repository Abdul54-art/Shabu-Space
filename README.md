### ✨ Key Features (คุณสมบัติเด่น)
* **📱 Responsive Design:** หน้าเว็บรองรับทุกอุปกรณ์ ทั้งมือถือ แท็บเล็ต และคอมพิวเตอร์
* **📊 Google Sheets Sync:** บันทึกข้อมูลการจองทันทีแม่นยำ 100%
* **🔒 Source Code Protection:** ใช้ระบบการเข้ารหัสไฟล์ (AES-128 Encryption) เพื่อป้องกันการเข้าถึงซอร์สโค้ดโดยไม่ได้รับอนุญาต

### 🔒 How to test (การเข้าชมผลงาน)
เนื่องจากโปรเจกต์นี้มีการป้องกันซอร์สโค้ด คุณสามารถทดสอบระบบได้โดย:
1. ดาวน์โหลดไฟล์ `run_app.py` และ `chabu_locked.bin`
2. รันคำสั่ง `python run_app.py`
3. กรอก **Access Key** ที่ได้รับจากผู้พัฒนาเพื่อเริ่มการใช้งาน
